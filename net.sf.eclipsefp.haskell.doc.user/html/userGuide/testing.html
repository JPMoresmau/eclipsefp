Testing

The recommended Haskell framework for tests is HTF. For documentation about how to use HTF, please refer to the official documentation.

To start a test right-click the project root, and select Run As > HTF test. The list of available test suites is shown, select the one you would like to run. The results are shown as they progress in the Haskell Test Results view.

There is also a HTF wizard in the Cabal editor. In the test suites section, you can create a new HTF test suite by indicating the source folder and the modules to test. A HTF module is created for each module tested, and a main module importing all of these is also created. A video of HTF integration can be found one YouTube

The plug-in can also execute Test-framework tests directly from the developer environment. For documentation about how to use Test-framework, please refer to the official documentation.

To start a test right-click the project root, and select Run As > Test-framework test. The list of available test suites is shown, select the one you would like to run. Once it is finished, the results are shown in the Haskell Test Results view.
