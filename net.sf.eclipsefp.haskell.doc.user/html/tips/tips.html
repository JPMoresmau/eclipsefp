Tips and Tricks

 

View compiler output

Compiling takes place in the background, when a source file is saved. Compiler errors and warnings are displayed in the Problems view, and as annotations in the source editors. You can also view some output of the Scion build process in Eclipse's Console view.
Default structure for Haskell projects

In the Preferences for the New Project wizard you can edit the default values for the source and documentation folders (Window > Preferences > Haskell > New Project). You can also let the project folder the source folder.
